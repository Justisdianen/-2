### 系统操作使用手册

#### 一、系统概述

**系统名称**: 若依管理系统
**系统版本**: 1.0.0
**系统类型**: 前后端分离管理系统
**技术栈**: SpringBoot3 + Vue3 + Element Plus + MySQL

**系统功能**:
- 用户管理：用户信息的增删改查、状态管理
- 角色管理：角色信息的增删改查、权限分配
- 菜单管理：菜单树形结构管理、权限控制
- 部门管理：部门树形结构管理

#### 二、登录系统

##### 2.1 访问系统

在浏览器中输入系统地址：
- 开发环境: `http://localhost:5173`
- 生产环境: `http://your-domain.com`

##### 2.2 登录页面

| 字段 | 说明 |
| :--- | :--- |
| 用户名 | 登录账号 |
| 密码 | 登录密码 |

##### 2.3 默认账号

| 用户名 | 密码 | 角色 |
| :--- | :--- | :--- |
| admin | admin123 | 超级管理员 |

##### 2.4 登录操作

1. 输入用户名和密码
2. 点击「登录」按钮
3. 登录成功后进入系统首页

#### 三、用户管理

##### 3.1 进入用户管理

1. 点击左侧菜单「系统管理」>「用户管理」
2. 进入用户列表页面

##### 3.2 用户列表

**功能说明**:
- 分页显示用户列表
- 支持按用户名、昵称、状态筛选
- 支持批量删除

**操作按钮**:
- 新增：添加新用户
- 编辑：修改用户信息
- 删除：删除单条用户记录
- 批量删除：删除选中的多条记录

##### 3.3 新增用户

1. 点击「新增」按钮
2. 填写用户信息：

| 字段 | 必填 | 说明 |
| :--- | :--- | :--- |
| 用户名 | 是 | 3-30字符，唯一 |
| 用户昵称 | 是 | 用户显示名称 |
| 密码 | 是 | 6-20字符 |
| 部门 | 否 | 所属部门 |
| 状态 | 否 | 默认正常 |

3. 点击「确定」保存

##### 3.4 编辑用户

1. 点击用户列表中的「编辑」按钮
2. 修改用户信息
3. 点击「确定」保存

##### 3.5 删除用户

**单条删除**:
1. 点击用户列表中的「删除」按钮
2. 确认删除操作

**批量删除**:
1. 勾选多条用户记录
2. 点击「批量删除」按钮
3. 确认删除操作

##### 3.6 状态管理

- 状态分为「正常」和「停用」
- 停用状态的用户无法登录系统
- 通过编辑用户可修改状态

#### 四、角色管理

##### 4.1 进入角色管理

1. 点击左侧菜单「系统管理」>「角色管理」
2. 进入角色列表页面

##### 4.2 角色列表

**功能说明**:
- 显示所有角色信息
- 支持按角色名称、状态筛选

##### 4.3 新增角色

1. 点击「新增」按钮
2. 填写角色信息：

| 字段 | 必填 | 说明 |
| :--- | :--- | :--- |
| 角色名称 | 是 | 角色显示名称 |
| 角色权限 | 是 | 权限标识，唯一 |
| 排序 | 否 | 显示顺序 |
| 状态 | 否 | 默认正常 |

3. 点击「确定」保存

##### 4.4 编辑角色

1. 点击角色列表中的「编辑」按钮
2. 修改角色信息
3. 点击「确定」保存

##### 4.5 删除角色

- 单条删除：点击「删除」按钮
- 批量删除：勾选多条记录后点击「批量删除」

##### 4.6 权限分配

1. 点击角色列表中的「权限分配」按钮
2. 在权限树中勾选该角色拥有的权限
3. 点击「确定」保存权限设置

#### 五、菜单管理

##### 5.1 进入菜单管理

1. 点击左侧菜单「系统管理」>「菜单管理」
2. 进入菜单列表页面（树形展示）

##### 5.2 菜单树形结构

**菜单类型**:
- 目录(M)：一级菜单，无页面
- 菜单(C)：二级菜单，有页面
- 按钮(F)：权限点，控制按钮显示

##### 5.3 新增菜单

1. 点击「新增」按钮
2. 填写菜单信息：

| 字段 | 必填 | 说明 |
| :--- | :--- | :--- |
| 菜单名称 | 是 | 菜单显示名称 |
| 上级菜单 | 否 | 父菜单ID |
| 排序 | 否 | 显示顺序 |
| 路由地址 | 否 | 页面路由路径 |
| 组件路径 | 否 | Vue组件路径 |
| 菜单类型 | 否 | M/C/F |
| 权限标识 | 否 | 权限字符串 |
| 菜单图标 | 否 | Element Plus图标 |
| 显示状态 | 否 | 是否显示在菜单中 |

3. 点击「确定」保存

##### 5.4 编辑菜单

1. 点击菜单列表中的「编辑」按钮
2. 修改菜单信息
3. 点击「确定」保存

##### 5.5 删除菜单

- 点击菜单列表中的「删除」按钮
- 确认删除操作
- 注意：有子菜单的菜单无法直接删除，需先删除子菜单

#### 六、部门管理

##### 6.1 进入部门管理

1. 点击左侧菜单「系统管理」>「部门管理」
2. 进入部门列表页面（树形展示）

##### 6.2 部门树形结构

- 支持多级部门嵌套
- 显示部门负责人和联系信息

##### 6.3 新增部门

1. 点击「新增」按钮
2. 填写部门信息：

| 字段 | 必填 | 说明 |
| :--- | :--- | :--- |
| 部门名称 | 是 | 部门显示名称 |
| 上级部门 | 否 | 父部门ID |
| 排序 | 否 | 显示顺序 |
| 负责人 | 否 | 部门负责人姓名 |
| 联系电话 | 否 | 部门联系电话 |
| 邮箱 | 否 | 部门邮箱 |
| 状态 | 否 | 默认正常 |

3. 点击「确定」保存

##### 6.4 编辑部门

1. 点击部门列表中的「编辑」按钮
2. 修改部门信息
3. 点击「确定」保存

##### 6.5 删除部门

- 点击部门列表中的「删除」按钮
- 确认删除操作
- 注意：有子部门或用户的部门无法直接删除

#### 七、权限控制说明

##### 7.1 菜单权限

- 根据用户角色显示对应的菜单
- 无权限的菜单不显示在左侧导航栏

##### 7.2 按钮权限

- 页面中的操作按钮根据权限控制显示/隐藏
- 无权限的用户看不到对应按钮

##### 7.3 数据权限

- 用户只能查看和操作自己部门的数据（如需）
- 超级管理员可查看所有数据

#### 八、系统设置

##### 8.1 修改密码

1. 点击右上角用户头像
2. 选择「修改密码」
3. 输入原密码和新密码
4. 点击「确定」保存

##### 8.2 退出登录

1. 点击右上角用户头像
2. 选择「退出登录」
3. 系统跳转到登录页面

#### 九、常见问题

##### 9.1 忘记密码

联系管理员重置密码

##### 9.2 菜单不显示

检查角色权限设置，确认该角色拥有对应菜单权限

##### 9.3 操作无权限

检查当前用户角色的权限配置

##### 9.4 数据无法删除

检查是否存在子级数据或关联数据

#### 十、操作规范

##### 10.1 数据录入规范

- 用户名：英文或数字，3-30字符
- 密码：6-20字符，建议包含大小写和数字
- 部门名称：中文，不超过30字符
- 角色名称：中文或英文，不超过30字符

##### 10.2 权限管理规范

- 最小权限原则：只分配必要的权限
- 定期审计：定期检查权限分配情况
- 离职处理：及时回收离职用户的权限

##### 10.3 数据备份

- 管理员定期备份数据库
- 重要操作前建议导出数据
