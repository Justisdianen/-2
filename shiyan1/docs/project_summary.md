### 项目复盘总结与问题日志

#### 一、项目概述

**项目名称**: 若依管理系统
**开发周期**: 2024年
**技术栈**: SpringBoot3 + Vue3 + Element Plus + MySQL

**完成功能**:
- 用户管理模块：CRUD、分页查询、状态管理、用户名唯一性校验
- 角色管理模块：CRUD、角色权限分配
- 菜单管理模块：树形结构、菜单类型控制、权限标识
- 部门管理模块：树形结构、祖级列表维护
- RBAC权限体系：菜单权限、按钮权限控制

#### 三、技术亮点

##### 3.1 后端技术亮点

1. **RBAC权限体系**: 基于角色的访问控制，支持菜单权限和按钮权限的细粒度控制

2. **MyBatis-Plus集成**: 使用代码生成器快速生成CRUD代码，减少重复开发

3. **Spring Cache缓存策略**: 使用Caffeine本地缓存，缓存树形结构数据，减少数据库查询

4. **全局异常处理**: 统一异常拦截，返回标准化错误信息

5. **敏感数据脱敏**: 请求参数自动脱敏处理，保护用户隐私

6. **参数校验**: 使用JSR-303注解进行参数校验，确保数据完整性

7. **逻辑删除**: 通过MyBatis-Plus逻辑删除，实现数据软删除

##### 3.2 前端技术亮点

1. **Vue3组合式API**: 使用Composition API组织代码，提高代码复用性

2. **Element Plus组件库**: 使用成熟的UI组件库，快速构建美观的管理界面

3. **路由懒加载**: 按需加载路由组件，优化首屏加载速度

4. **请求防抖与取消**: 避免重复请求，优化用户体验

5. **树形组件封装**: 封装通用树形组件，支持菜单和部门的树形展示

6. **响应式状态管理**: 使用Vue响应式API管理组件状态

7. **Vite构建工具**: 使用Vite作为构建工具，提供更快的开发体验

##### 3.3 架构设计亮点

1. **前后端分离**: 前端和后端完全分离，便于独立开发和部署

2. **RESTful API设计**: 遵循RESTful规范，接口设计清晰

3. **统一返回格式**: 所有接口返回统一格式，便于前端处理

4. **模块化设计**: 按业务模块划分代码结构，便于维护和扩展
